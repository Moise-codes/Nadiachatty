**Nadiachatty** is a premium, full-stack instant messaging platform designed for seamless real-time interaction. Featuring secure authentication, media sharing via Cloudinary, and high-performance state management, it provides a robust alternative to modern chat applications.

## 🔥 Enterprise-Grade Features

* **⚡ Real-Time Messaging:** Instant message delivery and "User Online" status via **Socket.io**.
* **🖼️ Media Cloud Integration:** Profile pictures and image sharing powered by **Cloudinary** and **Multer**.
* **🛡️ Hardened Security:** Implements **Helmet** for header protection, **Express-Rate-Limit** for DDoS mitigation, and **BcryptJS** for credential safety.
* **🧠 Global State Management:** Ultra-lightweight and reactive UI state using **Zustand**.
* **📧 Email Services:** Integrated **Nodemailer** for account verification and notifications.
* **🎨 Modern UI/UX:** Styled with **Tailwind CSS 4.0** and **DaisyUI** for a sleek, themeable interface.
